2. connection.queries

{% trans "Python list with all queries executed in the current request. Useful for programmatic inspection - in views, scripts or tests. Requires DEBUG=True." %} 

from django.db import connection, reset_queries

reset_queries()          # reset counter
qs = Book.objects.select_related("author")[:10]
list(qs)                 # force execution
print(len(connection.queries))   # count
print(connection.queries[0])     # {'sql': '...', 'time': '0.002'}

4. EXPLAIN ANALYZE (PostgreSQL)

{% trans "Shows the actual execution plan from PostgreSQL - identifies seq scans, used indexes, estimated cost and real time. Run directly in psql or via QuerySet.explain() in Django." %}
